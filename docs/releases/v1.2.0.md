# Synapse v1.2.0

Release date: 2026-02-15

## Added

- Configurable global hotkey presets (default `Option + Space`) with live update.
- AI translation capability (natural-language detection + `/translate`).
- Fuzzy slash matching with prefix/alias support (`/q` -> `/quit`).

## Changed

- Codex model strategy refactored to per-mode selection.
- Fast / Think / Deep Research each support independent reasoning effort.
- App open/close uses local app indexing + fuzzy matching (full app name no longer required).

## Fixed

- Improved slash prefix matching reliability.
- Better intent priority to avoid translation misrouting.
- Reduced open/close command strictness for app name matching.

## QA

- `swift build` passed.
- `swift test` passed.

- `Synapse-v1.2.0-macos-arm64.zip`

## Install

### Homebrew (custom tap)

```bash
brew tap 0smboy/synapse https://github.com/0smboy/Synapse
brew install --cask synapse-oboy
```

### Manual

1. Download `Synapse-v1.2.0-macos-arm64.zip`
2. Move `Synapse.app` to `/Applications`
3. Verify `codex --version`
