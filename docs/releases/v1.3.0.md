# Synapse v1.3.0

Release date: 2026-02-18

## Added

- Conversation history
  - AI execution output persisted to `conversation_history.json`
  - New `/history` query support with keyword filtering
- Memory layer
  - Persistent `memory.json`
  - New `/memory`, `/remember`, `/forget`
- Web search
  - New `/web` command (DuckDuckGo API baseline)
- Interaction upgrade
  - `open/close` flows support `↑/↓` candidate switching + Enter execution
- New voice edition
  - `SynapseVoice` (Headless AI Native)
  - Core tools: `read/write/edit/bash`
  - Filesystem memory/history
  - TTS output

## Changed

- AI prompts include Memory + Recent Conversation context.
- Matrix visual style introduced for AI cards (`matrix/native` switch).
- New settings:
  - `enableWebSearch`
  - `aiVisualStyle`

## QA

- `swift build` passed (`Synapse` + `SynapseVoice`)
- `swift test` passed
