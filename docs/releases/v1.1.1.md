# Synapse v1.1.1

Release date: 2026-02-15

## Added

- Menu bar Preferences now uses stable in-app native window management.
- README feature gallery updated to the latest 4 production screenshots.

## Changed

- Release assets simplified and no longer include image bundles.
- Core assets remain:
  - app zip
  - `Info.plist`
  - `SHA256SUMS.txt`

## Fixed

- Fixed menu bar Preferences no-op issue (`showSettingsWindow:` dependency removed).
- Preserved prior packaging fix for Codex executable path detection.

## Release Assets

- `Synapse-v1.1.1-macos-arm64.zip`
- `Info.plist`
- `SHA256SUMS.txt`

## Install

### Homebrew (custom tap)

```bash
brew tap 0smboy/synapse https://github.com/0smboy/Synapse
brew install --cask synapse-oboy
```

### Manual

1. Download `Synapse-v1.1.1-macos-arm64.zip`
2. Move `Synapse.app` to `/Applications`
3. Verify `codex --version`
